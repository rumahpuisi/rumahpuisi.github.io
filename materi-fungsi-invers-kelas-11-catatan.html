Maka dapat dituliskan : Jika fungsi  : A → B yang mempunyai peta (a) = b, maka invers  adalah fungsi g : B → A dengan peta g(b) = a Dapat dinyatakan dengan :      g  =  -1

22. materi fungsi (invers) kelas x jawab dengan cara yg lengkap dan benar untk belajar


kelas : X SMA
mapel : matematika
kategori ; fungsi
kata kunci : fungsi invers

kode : 10.2.6 [matematika SMA kelas 10 bab 6 fungsi]

Pembahasan:

f(x) = (2x - 3) / (x + 5)
f^-1(x) = ... ?

jawab:

(2x - 3) / ( x + 5) = y
2x - 3 = y (x + 5)
2x - 3 = xy + 5y
2x - xy = 5y + 3
x ( 2 - y) = 5y + 3
x = (5y + 3) / (2 - y)

jadi f^-1(x) = (5x + 3) / (2 - x), dan x ≠ 2

atau

y = (2x - 3) / (x + 5)
y (x + 5) = (2x - 3)
xy + 5y = 2x - 3
xy - 2x = -5y -3
x (y - 2) = -5y - 3
x = (-5y -3) / (y - 2)

jadi f^-1(x) = (-5x -3) / (x - 2), dan x ≠ 2

39. cara invers dong yang ngerti kelas 11 SMA


Invers sendiri sebenarnya dapat digolongkan kedalam beberapa materi (Fungsi, matriks, dll). Berikut saya jelaskan tentang invers dari suatu matriks.
Dimisalkan terdapat sebuah matriks A: [tex] \left[\begin{array}{ccc}a&b\\c&d\end{array}\right][/tex]
Maka, inversnya adalah dengan cara: [tex]\frac{1}{detA} \left[\begin{array}{ccc}a&b\\c&d\end{array}\right] [/tex]
Determinan didapat dari (axb) - (cxd). Hasil dari determinan tersebut (Misalkan hasil determinannya 2), maka matriks A tadi semua anggotanya dikalikan dengan [tex]\frac{1}{2}[/tex], sehingga menjadi:
=> [tex]\frac{1}{2} \left[\begin{array}{ccc}a&b\\c&d\end{array}\right] [/tex]
=> [tex] \left[\begin{array}{ccc}\frac{a}{2}&\frac{b}{2}\\\frac{c}{2}&\frac{d}{2}\end{array}\right] [/tex]
