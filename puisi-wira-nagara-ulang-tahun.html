= (angsuran selama 1 tahun - bunga nominal) ÷ lama angsuran

= 3.840.000 ÷ 1
= 3.840.000

40. Rata-rata nilai ulangan 4 orang siswa adalah 75. Jika nilai Kadek Wira dimasukkan, maka rata-rata nilai mereka 80. Nilai ulangan Kadek Wira adalah ....tolong besok harus dikumpulin​


Jawaban:

x = -10

Penjelasan dengan langkah-langkah:

Misalkan nilai-nilai ulangan siswa yang sudah diketahui adalah A, B, dan C, dan nilai ulangan Kadek Wira adalah x. Maka, berdasarkan informasi soal, kita dapat membuat persamaan rata-rata sebagai berikut:

(75 + A + B + C) / 4 = 80

Kita dapat menyederhanakan persamaan tersebut dengan cara mengalikan kedua ruas dengan 4:

75 + A + B + C = 320

Selanjutnya, kita dapat mencari nilai x (nilai ulangan Kadek Wira) dengan mengganti A, B, dan C dengan nilai-nilai yang diketahui, dan menyelesaikan persamaan di atas:

75 + 80 + 85 + 90 + x = 320

330 + x = 320

x = 320 - 330

x = -10

Jadi, nilai ulangan Kadek Wira adalah -10. Namun, perlu diperhatikan bahwa nilai ulangan tidak dapat bernilai negatif, sehingga jawaban ini tidak masuk akal. Mungkin ada kesalahan dalam masalah atau informasi yang diberikan.
