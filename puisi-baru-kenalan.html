LIM x→∞ (2 + 4/x² - 1/x³) / (5 - 2/x)
untuk nilai x→∞, maka tiap bentuk /x akan menjadi nol
sehingga tinggal (2 + 0 - 0)/(5 - 0)
jadi jawabannya = 2/5 (A)

19. gunakan bentuk
LIM x→∞ √(ax² + bx + c) - √(px² + qx + r) = (b-q) / 2√a
pada soal tersebut, a = 1 , b = 2 , q = 3
sehingga jawabannya = (2 - 3)/2√1 = -1/2
