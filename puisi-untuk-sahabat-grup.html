Jawaban:

1. Grup Alignment  

2. Grup Number

3. Grup Cells

4. Grup Illustrations

5. Grup Charts

6. Grup Page Setup

7. Grup  Scale to Fit

8. Grup Function Library

9. Grup Sort & Filter

10. Grup Zoom

Penjelasan:

Misalkan pada Ms. Excel 2013:

- Grup Alignment, Number dan Cells ada dalam Ribbon Home.

- Grup Illustrations dan Charts ada dalam Ribbon Insert.

- Grup Page Setup dan Scale to Fit ada dalam Ribbon Page Layout.

- Grup Function Library ada dalam Ribbon Formulas.

- Grup Sort & Filter ada dalam Ribbon Data.

- Grup Zoom ada dalam Ribbon View.

Khusus untuk nomor 1,2 dan 3 bisa melihat gambar berikut ini:

- Grup Alignment, Number dan Cells ditunjukkan tanda panah merah

- Ribbon Home ditunjukkan tanda panah biru.
